## 1.确定高层次的解决方案

| 要素 | 内容 |
| --- | --- |
| ID | P4 |
| 解决方案S1 |
| 方案描述 | 系统可以提供建立文档类别的功能 |
| 业务优势 | 可以自己定义文档类别 |

| 方案描述 | 系统自动将文档归类 |
