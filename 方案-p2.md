## 1.确定高层次的解决方案

| 要素 | 内容 |
| --- | --- |
| ID | S1 |
| 解决方案S1 |
| 方案描述 | 引入文件垃圾箱，可以找回删除的文件 |
| 业务优势 | 能够有效降低误删文档带来的风险|
| 代价 | 消耗了本可以释放的存储空间 |
| 解决方案S2 |
| 方案描述 | 引入用户权限，只有指定的用户才可以删除文件 |
| 业务优势 | 能够有效降低误删文件带来的风险|
| 代价 | 增加了有删除权限的用户负担 |


## 2.确定系统特性和解决方案的边界

### 选择：解决方案S1

#### (1)系统特性

SF1：删除文件时自动将该文件的所有版本缓存

SF2：删除文件的单个版本时会将该版本的文件缓存

SF3: 找回文件时还可以找回相应的版本信息
