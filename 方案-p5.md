# 方案-P5

## 确定高层次的解决方案

| 要素 | 内容 |
| --- | --- |
| ID | P5 |
| 解决方案1 |
| 方案描述 | 便捷的任务管理，系统记录每次活动中成员的任务分工，并且提醒用户当前任务，提供实时的任务进度详情，可以及时查看自己的任务内容和任务进度。 |
| 业务优势 | 及时查看自己的任务内容和进度，提高工作效率。 |
| 代价 | 无 |
| 解决方案2 |
| 方案描述 | 用户自助的任务提醒机制。项目PM通过系统向成员告知分工内容，根据进度提醒团队成员。 |
| 业务优势 | 可以直接高效地进行用户间的进度确认。 |
| 代价 | 如果PM没有及时参与，会造成任务执行效率低下。 |
